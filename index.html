Java Examinator

This program, is a simple program that simulates the JAVA Certification Examination. First, you would have to provide your user id and your four-digit password to start the exam. If the user successfully logs in, a preparation message will pup up to inform the user that he/she has 10 minutes to answer 20 questions about JAVA. The exam will start as soon as the user presses ENTER. The user id, password, and the number of exams taken by an user are stored in a database. All the questions that the program uses are also stored in a database. All questions are devided in four levels of difficulty, level one the lowest and level four the highest. Click here and take a look at its GitHub repo.

  • Date: December 2010
